# T3dassonne

A cozy, browser-based tile-laying kingdom game. Draw a tile, place it to extend the
map, and set your followers on roads, cities, cloisters and farms to score points.
Play against the computer or pass-and-play locally on one device.

**▶ Play:** https://t3dboy.github.io/t3dassonne/

- Base game (72 tiles) plus the optional River expansion
- 2–5 players, three AI difficulties
- Hand-drawn tile art rendered to canvas, animated final scoring
- No dependencies, no tracking — a single static page

## Develop

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

The production site is a static bundle in `dist/`, deployed to GitHub Pages on every
push to `main`.
